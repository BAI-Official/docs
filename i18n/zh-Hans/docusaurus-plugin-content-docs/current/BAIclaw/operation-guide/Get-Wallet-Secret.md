# 获取 Web3 钱包私钥

私钥是你在区块链上控制资产的唯一凭证。在配置 BAIclaw Agent Wallet 时，你需要导入与 BAI API Key 关联的钱包私钥。本文将指导你从常见的 Web3 钱包中安全地导出私钥。

:::danger[安全警告]
**私钥 = 资产的完全控制权。** 任何持有你私钥的人都可以直接转移你的全部资金。请像对待银行密码一样对待你的私钥——**永远不要**向任何人透露。
:::

## 开始之前

在导出私钥前，请确认以下安全条件已满足：

- ✅ 你正处于一个**安全、私密的环境**，周围没有其他人可以看到你的屏幕
- ✅ 你的电脑**没有被远程控制**（关闭 TeamViewer、向日葵等远程软件）
- ✅ 你的设备**没有感染恶意软件**（建议提前进行一次安全扫描）
- ✅ 你不会通过截图、聊天工具或邮件发送私钥

:::tip[建议]
导出私钥后，建议立即粘贴到 BAIclaw 的 Agent Wallet 配置中，**不要**将私钥保存在剪贴板、记事本或任何文件中。完成配置后，清空剪贴板内容。
:::

## 各钱包导出指南

### TronLink

1. 打开 **TronLink** 浏览器插件
2. 点击页面左上角的**钱包详情**
3. 在菜单中选择 **导出钱包/私钥**
4. 输入钱包密码进行验证
5. 复制显示的私钥

### MetaMask

1. 打开 **MetaMask** 浏览器插件
2. 点击左上角的**账户**图标
3. 在目标账户右侧点击 **三个点（⋮）**，选择 **"账户详情"**
4. 点击 **"显示私钥"**
5. 输入钱包密码进行验证
6. 复制显示的私钥

### OKX Wallet

1. 打开 **OKX Wallet** 浏览器插件
2. 点击左上角的**钱包管理图标**（或进入 **设置 → 钱包管理**）
3. 在当前钱包右侧点击 **三个点（⋮）** 或 **"管理"** 图标
4. 选择 **"备份钱包"** 或 **"查看私钥"**
5. 输入钱包密码进行验证
6. 复制显示的私钥

### Binance Wallet

1. 打开 **Binance Wallet** 浏览器插件
2. 点击右上角的 **设置（齿轮）** 图标
3. 选择 **"账户管理"**
4. 点击 **"导出私钥"**
5. 输入钱包密码进行验证
6. 复制显示的私钥

### TokenPocket

1. 打开 **TokenPocket** 浏览器插件
2. 点击左上角的**钱包图标**进入详情页
3. 点击 **"详情"** 或 **"钱包管理"**
4. 找到 **"导出私钥"** 选项
5. 输入钱包密码进行验证
6. 复制显示的私钥

:::info[关于密码验证]
几乎所有钱包在导出私钥时都会要求输入密码。这是钱包的安全保护机制——如果你忘记了钱包密码，需要先通过助记词恢复钱包，再重新设置密码后导出。
:::

## 导出后的安全建议

完成私钥导出并配置 Agent Wallet 后，请执行以下操作：

1. **清空剪贴板** — 复制一段无关文字覆盖剪贴板中的私钥内容
2. **检查浏览器** — 确保没有扩展程序或网页记录了你的私钥
3. **不要截图保存** — 截图可能被云同步服务自动上传

## 常见问题

**Q: 为什么 BAIclaw 需要我的私钥？**
A: BAIclaw 的 Agent Wallet 需要私钥来代表你签名和执行链上交易。私钥仅在本地加密存储，不会上传到任何服务器。

**Q: 我的钱包不在上述列表中，怎么办？**
A: 大多数 Web3 钱包都提供私钥导出功能，通常在 **设置 → 安全 → 导出私钥** 路径下。如果找不到，请查阅你所使用钱包的官方文档。

**Q: 导出的私钥是什么格式？**
A: 私钥通常是一串 64 位的十六进制字符串（如 `a1b2c3d4...`）。

**Q: 私钥和助记词有什么区别？**
A: 助记词（通常为 12 或 24 个英文单词）。在 BAIclaw 中，你只需要导入单个私钥即可。

**Q: 导出私钥后，原钱包还能正常使用吗？**
A: 可以。导出私钥只是"查看"操作，不会影响原钱包的任何功能。你的原钱包和 Agent Wallet 将共享同一个地址和资产。
