# BAIclaw

## 简介

**BAIclaw** 是一款基于 [OpenClaw](https://openclaw.ai/) 和 [ClawX](https://github.com/ValueCell-ai/ClawX/blob/main/README.md) 打造的个人 AI 智能体应用。BAIclaw 继承了 OpenClaw 的核心能力，并为用户提供了开箱即用的体验和图形化操作界面。

更多使用技巧请参考 [OpenClaw 文档](https://docs.openclaw.ai/)

---

## 核心特性

### 🎯 开箱即用

全程通过图形用户界面（GUI）完成从安装到对话的全部配置，无需执行终端命令或手动编写配置文件。

### 💬 智能交互界面

- **沉浸式体验：** 支持多会话上下文管理、历史记录留存及 Markdown 富文本渲染。
- **Agent 直接路由：** 支持在主输入框通过 `@agent` 指令实现精准路由。切换后将直接进入目标 Agent 的独立对话上下文，而非通过默认 Agent 转发。

### 📡 频道管理

- **独立运行：** 支持配置多个独立运行的 AI 频道，以适配特定任务流。
- **账号绑定：** 每个频道支持多账号管理。用户可在 Channels 页面直接完成 Agent 账号绑定与默认账号切换。

### ⏰ 自动化调度

可自定义触发器与时间间隔，通过预设定时任务，实现 AI Agent 7×24 小时无间断自动化执行。

### 🧩 技能扩展系统

- **免包管理器：** 内置集成技能面板，支持技能的浏览、安装与管理。
- **预置技能：** 启动时自动向托管目录（默认 `~/.openclaw/skills`）部署文档处理技能（PDF、XLSX、DOCX、PPTX）及常用搜索技能。
- **来源追踪：** 自动识别并展示来自托管目录、工作区及额外目录的技能来源，支持直接跳转至技能物理安装路径。

### ⚙️ 系统设置

- **自适应主题：** 适配浅色、深色模式或跟随系统全局设置。
- **开机自启动：** 支持在【设置 → 通用】中配置系统登录后自动运行。

## 快速上手

### 系统要求

- **操作系统：** macOS 11+
- **内存：** 最低 4GB RAM（推荐 8GB）
- **存储空间：** 1GB 可用磁盘空间

### 首次启动

首次启动 BAIclaw 时，**设置向导**将引导你完成以下步骤：

1. **语言与区域** – 配置你的首选语言和地区
2. **添加 BANK OF AI 模型** – 填写 BANK OF AI 的 API Key 来获得顶级大模型能力（详见：[获取 API Key](./Get-API-Key.md)）
3. **技能包** – 选择适用于常见场景的预配置技能
4. **验证** – 在进入主界面前测试你的配置

## 核心页面

| 页面 | 说明 |
|------|------|
| **对话** | 与 AI 智能体进行实时聊天的核心界面，支持多轮对话、消息渲染和输入交互 |
| **模型** | 配置和管理 AI 模型供应商（OpenAI、Claude、Gemini 等），设置默认模型和 API Key |
| **Agents** | 添加和管理多个 AI 智能体 |
| **频道** | 连接和管理外部通讯平台（Telegram、Discord 等），实现消息收发 |
| **技能** | 安装和管理 AI 技能/插件，扩展智能体的能力（如浏览器控制、文件处理等） |
| **定时任务** | 创建和管理自动化定时任务，周期性触发 AI 执行工作流，实现 7×24 小时不间断运行 |
| **设置** | 应用全局配置，包括外观、通知、代理等高级选项 |
