# 获取 API Key

请按照以下步骤获取 BANK OF AI 的 **API Key**:

1. 登录 [BANK OF AI](https://chat.bankofai.io/chat)
2. 访问侧边栏中的 API 功能，在页面中点击 **创建API Key**
3. 复制创建的 **API Key** 到 BAIclaw 中

![](./assets/Pasted%20image%2020260317232256.png)

:::danger[请妥善保管你的 API Key]
- API Key 是你调用 BANK OF AI 模型接口的重要凭证，请勿将其泄露给第三方。
- 一旦泄露，第三方可能使用你的配额进行接口调用，导致你的账号产生超额费用，相关损失将由账号持有者自行承担。
- 如果发现 API Key 可能泄漏，请删除该 API Key，重新创建并使用新的 API Key。
:::
