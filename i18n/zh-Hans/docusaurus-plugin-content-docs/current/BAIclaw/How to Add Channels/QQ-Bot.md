# QQ Bot

## 前提

- QQ 账号

## 配置步骤

### 1. 创建机器人

1. 进入 [快速注册创建 QQ 机器人](https://q.qq.com/qqbot/openclaw/login.html)
2. 点击**创建机器人**
3. 在机器人页面中找到 **AppID** 和 **AppSecret**，分别点击右侧复制按钮，保存到记事本或备忘录中
   > **注意：** **AppSecret** 不支持明文保存，离开页面后再查看会强制重置，请务必妥善保存。
