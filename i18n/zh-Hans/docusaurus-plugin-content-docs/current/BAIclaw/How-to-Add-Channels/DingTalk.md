# 钉钉

## 前提

- 钉钉账号

## 配置步骤

### 1. 创建企业

1. 点击**创建或加入企业**

![](./assets/Pasted%20image%2020260319004613.png)

2. 点击"企业"后，选择**创建企业/团体**

![](./assets/Pasted%20image%2020260319005014.png)

3. 填写信息并创建（"企业名称"需要超过三个字，其他内容可随意填写）

![](./assets/Pasted%20image%2020260319005321.png)

### 2. 创建机器人

前往 [钉钉开发者平台](https://open-dev.dingtalk.com)，选择刚刚创建的组织。

![](./assets/Pasted%20image%2020260319005820.png)

1. 选择**创建**

![](./assets/Pasted%20image%2020260319010139.png)

2. 左侧栏目选择"机器人"后，点击**创建应用**

![](./assets/Pasted%20image%2020260319010213.png)

3. 填写"应用名称"和"应用描述"后点击**保存**

![](./assets/Pasted%20image%2020260319010557.png)

4. 在左侧栏目选择**添加应用能力**，选择添加"机器人"

![](./assets/Pasted%20image%2020260319010802.png)

5. 开启机器人配置并填写内容

![](./assets/Pasted%20image%2020260319010941.png)

![](./assets/Pasted%20image%2020260319011156.png)

### 3. 调整权限并发布

1. 左侧栏目选择"权限管理"，搜索"消息"，勾选图中两项后点击**批量申请**

![](./assets/Pasted%20image%2020260319011512.png)

2. 左侧栏目选择"版本管理与发布"，点击**创建新版本**（必须创建版本信息才能发布）

![](./assets/Pasted%20image%2020260319011628.png)

3. 输入"应用版本号"和"版本描述"后**保存**，即发布完成
4. 选择"凭证与基础信息"，复制 **APP ID** 和 **Client Secret** 到 BAIclaw 中保存并连接

### 4. 搜索并使用机器人

在钉钉**消息**栏的搜索框中搜索刚创建的"机器人名称"，会展示在功能栏下。
