# Telegram

## 前提

- Telegram 账号

## 配置步骤

### 1. 创建 Bot 并获取 Bot Token

1. 打开 Telegram，搜索并打开 **@BotFather**（带认证）
2. 发送 `/newbot`
3. 按提示输入并创建 bot（**必须以 `bot` 结尾**，例如 `myclaw123bot`）
4. 成功后复制 **Bot Token**（格式：`123456:ABCdef...`）并粘贴到 BAIclaw 的**机器人令牌**输入框中

![](./assets/Pasted%20image%2020260319003834.png)

### 2. 获取用户 ID

1. 搜索并打开 **@userinfobot** 并点击开始
2. 从 @userinfobot 获取您的用户 ID 并粘贴到 BAIclaw 的**允许的用户 ID** 输入框中
3. 在 Telegram 搜索你刚创建的 Bot 并进行对话
