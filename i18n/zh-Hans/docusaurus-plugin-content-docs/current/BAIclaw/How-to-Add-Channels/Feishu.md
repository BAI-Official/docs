# 飞书

## 配置步骤

1. 打开链接 [一键创建机器人](https://open.feishu.cn/page/openclaw?form=multiAgent)
2. 复制 **ID**、**Secret** 到 BAIclaw 配置页面即可完成配置
3. 更多信息请参考：[OpenClaw 飞书官方插件使用指南（公开版）](https://bytedance.larkoffice.com/docx/MFK7dDFLFoVlOGxWCv5cTXKmnMh)
