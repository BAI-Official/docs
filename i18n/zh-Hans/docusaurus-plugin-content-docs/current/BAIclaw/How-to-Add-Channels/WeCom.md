# 企业微信

## 前提

- 企业微信客户端（需升级到最新版本）

## 配置步骤

### 1. 创建机器人

1. 在客户端 → 工作台，点击**智能机器人** → **创建机器人**，选择 **API 模式**创建

![](./assets/Pasted%20image%2020260319013330.png)

![](./assets/Pasted%20image%2020260319013354.png)

2. 选择以「长连接」方式创建，并获取 **Bot ID** 和 **Secret**

![](./assets/Pasted%20image%2020260319013614.png)

### 2. 在 BAIclaw 中添加企业微信 Bot

将获得的 **Bot ID** 和 **Secret** 填入 BAIclaw 中。
