# Get API Key

Follow these steps to get your BANK OF AI **API Key**:

1. Log in to [BANK OF AI](https://chat.bankofai.io/chat)
2. Go to the API feature in the sidebar, click **Create API Key** on the page
3. Copy the created **API Key** to BAIclaw

![](./assets/Pasted%20image%2020260317232256.png)

:::danger[Please keep your API Key safe]
- The API Key is an important credential for calling the BANK OF AI model interface. Do not share it with third parties.
- Once leaked, third parties may use your quota for interface calls, causing excess charges on your account. Any related losses will be borne by the account holder.
- If you suspect your API Key has been compromised, delete that API Key, recreate it, and use the new API Key.
:::
